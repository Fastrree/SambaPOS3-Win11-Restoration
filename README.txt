============================================================
[EN] IMPORTANT NOTES AFTER INSTALLATION
============================================================

1. RUN AS ADMINISTRATOR: 
   To prevent database connection errors, please right-click on 
   the SambaPOS shortcut on your desktop and select 
   "Run as Administrator".

2. SQL SERVER CONNECTION: 
   SambaPOS V3 requires SQL Server 2008 R2 or 2012. If the 
   application fails to start, ensure your SQL Instance is 
   active and correctly configured in settings.

3. SCREEN RESOLUTION: 
   If the buttons look too small or misaligned on Windows 11, 
   go to Compatibility settings and enable "High DPI scaling override".

Thank you for using SambaPOS V3 (Windows 11 Compatible Version).

1. YÖNETİCİ OLARAK ÇALIŞTIRIN: 
   Veritabanı bağlantı hatalarını önlemek için masaüstündeki
